# final_project
My final project for Machine Learning. 

## Project Overview

This project uses clustering methods to identify student “AP personas” based on patterns in the data. The goal was to explore whether students could be grouped into meaningful profiles that reflect differences in AP participation and performance. The project compares clustering approaches to better understand which method produced the most useful and interpretable groups.

## How to Run the Project

All code for data cleaning, clustering, model comparison, and interpretation is contained in the Python notebook file. Open the `.ipynb` notebook and run the cells from top to bottom to reproduce the analysis.
